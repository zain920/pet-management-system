PET MANAGEMENT SYSTEM
This is a pet record management system that stores and manages information about pets.

MAIN FEATURES:
• Add new pets with their ID, name, type (like dog/cat/bird), and age. 
• View all pets stored in the system • Search for specific pets by their name 
• Update existing pet information (change type or age) 
• Delete pets from the system 
• Search for all pets of a particular type (find all dogs, all cats, etc.) 
• Save pet data to a text file that you can read
• Save pet data to a binary file for efficient storage 
• Load previously saved pet data from binary files

SYSTEM SPECIFICATIONS:
• Simple menu-based navigation system 
• Capacity: Up to 100 pets
• Data storage: In-memory during operation 
• File persistence: Optional saving to text/binary files
